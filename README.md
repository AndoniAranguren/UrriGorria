# UrriGorria
A game of sink the fleet made in Java that supports player vs machine, player vs player, and players(1 to 49) vs machines (1 to 49).
Game loop is:
1. Place your fleet
2. Buy an item in the shop
3. Use utility items
4. Use weapons
5. Check you are the last player standing
6. Go back to step 2
